# Top Edge - Website Mockup

Website concept for Top Edge, a premium window tinting shop in Clovis, CA.

- **Location:** 1010 San Jose Ave Suite 101, Clovis, CA
- **Phone / WhatsApp:** (559) 668-3485
- **Instagram:** [@_top_edge_](https://instagram.com/_top_edge_)
- **Existing site:** [topedgepro.com](https://topedgepro.com) (GoDaddy builder)

Built by [Rift Media](https://riftmedia.cc).

## Stack

Single-file static HTML + CSS + vanilla JS. No build step. Hosted on Vercel.

Highlights:
- 2-step visual quote configurator (clickable SVG car diagram + shade picker)
- Phone-formatted inputs, SMS fast-path, 3-field contact form
- Sticky mobile call/WhatsApp/quote bar
- Portrait gallery grid pulling from Top Edge's existing image CDN

## Local preview

Open `index.html` directly in a browser.
